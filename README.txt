This is a Simple Site about the Cactuses World on Django framework.
Project: Python, Django 
 
 Site functionality:
 1. Start page: 
 • list of all registered cacti of different users 
 • cacti registration form 
 • links to the registration and authentication / authorization pages of the user
 2.  Access to the registered user profile: 
 • displays all registered user cacti 
 3.  The administrative part: 
 • the ability to add / delete / edit cacti
 • the ability to select a user for a particular cactus
 4.  Cactus card: 
 • contains information about a particular Cactus 
 5.  The project is internationalized.
